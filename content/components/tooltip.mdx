# Tooltip Component

Tooltips are used to provide additional information when hovering over elements.

## Basic Usage

<EnhancedCodePreview
  componentName="Tooltip"
  code={`import { Tooltip } from '@/data/components/Tooltip/Tooltip';
import { Button } from '@/data/components/Button/Button';

// Basic tooltip
<Tooltip content="This is a tooltip">
  <Button>Hover me</Button>
</Tooltip>

// With custom content
<Tooltip content={<div>Custom <strong>content</strong></div>}>
  <Button>Hover me</Button>
</Tooltip>`}
>
  <div className="flex flex-wrap gap-4 items-center">
    <Tooltip content="This is a tooltip">
      <Button>Hover me</Button>
    </Tooltip>
    <Tooltip content={<div>Custom <strong>content</strong></div>}>
      <Button>Hover me</Button>
    </Tooltip>
  </div>
</EnhancedCodePreview>

## Delay

Tooltips can have a delay before appearing:

<EnhancedCodePreview
  componentName="TooltipDelay"
  code={`<div className="flex flex-wrap gap-4 items-center">
  <Tooltip content="No delay" delay={0}>
    <Button>No delay</Button>
  </Tooltip>
  <Tooltip content="500ms delay" delay={0.5}>
    <Button>500ms delay</Button>
  </Tooltip>
  <Tooltip content="1000ms delay" delay={1}>
    <Button>1000ms delay</Button>
  </Tooltip>
</div>`}
>
  <div className="flex flex-wrap gap-4 items-center">
    <Tooltip content="No delay" delay={0}>
      <Button>No delay</Button>
    </Tooltip>
    <Tooltip content="500ms delay" delay={0.5}>
      <Button>500ms delay</Button>
    </Tooltip>
    <Tooltip content="1000ms delay" delay={1}>
      <Button>1000ms delay</Button>
    </Tooltip>
  </div>
</EnhancedCodePreview>
