## Customization

### Autoplay

Enable or disable autoplay and adjust the interval:

```tsx
<FeatureTabs tabs={tabs} autoplay interval={4000} />
```

### Layout height

Control the preview area height with the `height` prop:

```tsx
<FeatureTabs tabs={tabs} height="min-h-[420px]" />
```
