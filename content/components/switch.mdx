# Switch Component

A highly customizable and accessible toggle component with built-in support for dark mode, size variations, visual variants, and smooth micro-interactions.

## Preview

  <div className="flex flex-wrap gap-4 items-center">

    <Switch variant="glass" color="success" />

  </div>
</ComponentPreview>

## Basic Usage

<EnhancedCodePreview
  componentName="Switch"
  code={`import { Switch } from '@/data/components/Switch/Switch';

// Basic switch
<Switch />

// Controlled switch
<Switch checked={true} onChange={(checked) => console.log(checked)} />

// Disabled switch
<Switch disabled />

// With accessible label
<Switch label="Enable notifications" />

// Screen-reader only label
<Switch srLabel="Dark mode toggle" />`}
>
  <div className="flex flex-wrap gap-4 items-center">
    <Switch />
    <Switch checked />
    <Switch disabled />
    <Switch label="Enable notifications" />
    <Switch srLabel="Dark mode toggle" />
  </div>
</EnhancedCodePreview>

## Variants

You can customize the visual style using the `variant` prop:

- `default` (neumorphic)
- `flat`
- `glass`
- `outline`
- `pill`
- `solid`

## Micro-interactions

The Switch includes rich micro-interactions:
- Smooth spring-based knob animation
- Background highlight on active
- Focus-visible ring for accessibility
- Neumorphic and glassmorphic visual variants
