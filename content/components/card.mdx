# Card Component

Card components are used to group related content and actions. This component includes micro-interactions and animations for better user experience.

## Basic Usage

<EnhancedCodePreview
  componentName="Card"
  code={`import { Card, CardHeader, CardContent, CardFooter } from '@/data/components/Card/Card';

// Basic card
<Card>
  <CardHeader title="Card Title" description="Card description" />
  <CardContent>
    <p>Card content goes here</p>
  </CardContent>
  <CardFooter>
    <Button>Action</Button>
  </CardFooter>
</Card>`}
>
  <Card>
    <CardHeader title="Card Title" description="Card description" />
    <CardContent>
      <p>Card content goes here</p>
    </CardContent>
    <CardFooter>
      <Button>Action</Button>
    </CardFooter>
  </Card>
</EnhancedCodePreview>

## Card Parts

Cards are composed of several parts:

- CardHeader: For the title and description
- CardContent: For the main content
- CardFooter: For actions and additional content

<EnhancedCodePreview
  componentName="CardParts"
  code={`<Card>
  <CardHeader 
    title="Card Title" 
    description="This is a description of the card"
  />
  <CardContent>
    <p>This is the main content of the card. It can contain any type of content, including text, images, forms, or other components.</p>
  </CardContent>
  <CardFooter>
    <Button variant="outline">Cancel</Button>
    <Button>Save</Button>
  </CardFooter>
</Card>`}
>
  <Card>
    <CardHeader 
      title="Card Title" 
      description="This is a description of the card"

    </CardContent>
    <CardFooter>
      <Button variant="outline">Cancel</Button>
      <Button>Save</Button>
    </CardFooter>
  </Card>
</EnhancedCodePreview>
