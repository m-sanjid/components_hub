# Checkbox Component

Checkbox components are used to toggle between checked and unchecked states.

## Basic Usage

<EnhancedCodePreview
  componentName="Checkbox"
  code={`import { Checkbox } from '@/data/components/Checkbox/Checkbox';

// Basic checkbox
<Checkbox label="Accept terms and conditions" />

// With onChange handler
<Checkbox 
  label="Subscribe to newsletter" 
  onChange={(checked) => console.log(checked)} 
/>

// Checked by default
<Checkbox label="Remember me" checked />`}
>
  <div className="flex flex-col gap-4">
    <Checkbox label="Accept terms and conditions" />
    <Checkbox 
      label="Subscribe to newsletter" 
      onChange={(checked) => console.log(checked)} 
    />
    <Checkbox label="Remember me" checked />
  </div>
</EnhancedCodePreview>

## States

Checkboxes can be in different states:

- Default
- Checked
- Error
- Disabled

<EnhancedCodePreview
  componentName="CheckboxStates"
  code={`<div className="flex flex-col gap-4">
  <Checkbox label="Default state" />
  <Checkbox label="Checked state" checked />
  <Checkbox label="Error state" error="This field is required" />
  <Checkbox label="Disabled state" disabled />
</div>`}
>
  <div className="flex flex-col gap-4">
    <Checkbox label="Default state" />
    <Checkbox label="Checked state" checked />
    <Checkbox label="Error state" error="This field is required" />
    <Checkbox label="Disabled state" disabled />
  </div>
</EnhancedCodePreview>
