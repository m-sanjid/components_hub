# Animated Accordion Component

The Animated Accordion component provides a beautiful way to display expandable content sections with smooth animations. It supports both single and multiple item expansion modes, custom icons, and disabled states.

## Key Features

- **Smooth Animations**: Built with Framer Motion for fluid transitions
- **Multiple Modes**: Support for single and multiple item expansion
- **Custom Icons**: Add icons to accordion items
- **Disabled States**: Disable specific accordion items
- **Responsive Design**: Works seamlessly across all device sizes
- **Accessibility**: Keyboard navigation and screen reader support
- **Customizable**: Easy to style and extend with your own content

## Customization Options

### Accordion Item Structure

```typescript
interface AccordionItem {
  id: string;
  title: string;
  content: React.ReactNode;
  icon?: React.ReactNode;
  disabled?: boolean;
}
```

### Available Props

- `items`: Array of AccordionItem objects
- `type`: Expansion mode ("single" | "multiple")
- `defaultOpen`: Array of item IDs to be expanded by default
- `className`: Additional CSS classes for styling

### Animation Configuration

Built with Framer Motion, the component includes:

- **Smooth Height Transitions**: Fluid animations for expanding/collapsing
- **Rotation Animation**: Smooth rotation of the chevron icon
- **Opacity Transitions**: Fade in/out effects for content
- **Gesture Support**: Touch-friendly interactions

## Accessibility Features

- **Keyboard Navigation**: Full keyboard support for expanding/collapsing
- **Screen Reader Friendly**: Proper ARIA labels and semantic HTML
- **Focus Management**: Clear focus indicators for interactive elements
- **Disabled States**: Proper handling of disabled items
- **High Contrast**: Ensures adequate color contrast for all states
