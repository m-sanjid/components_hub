# Radio Component

Radio components are used to select a single option from a list of options.

## Basic Usage

<EnhancedCodePreview
  componentName="Radio"
  code={`import { Radio } from '@/data/components/Radio/Radio';

// Basic radio
<Radio value="option1" label="Option 1" />

// With onChange handler
<Radio 
  value="option2" 
  label="Option 2" 
  onChange={(value) => console.log(value)} 
/>

// Checked by default
<Radio value="option3" label="Option 3" checked />`}
>
  <RadioOption3 />
</EnhancedCodePreview>

## States

Radio buttons can be in different states:

- Default
- Checked
- Error
- Disabled

<EnhancedCodePreview
  componentName="RadioStates"
  code={`<div className="flex flex-col gap-4">
  <Radio value="default" label="Default state" />
  <Radio value="checked" label="Checked state" checked />
  <Radio value="error" label="Error state" error="This field is required" />
  <Radio value="disabled" label="Disabled state" disabled />
</div>`}
>
  <div className="flex flex-col gap-4">
    <Radio value="default" label="Default state" />
    <Radio value="checked" label="Checked state" checked />
    <Radio value="error" label="Error state" error="This field is required" />
    <Radio value="disabled" label="Disabled state" disabled />
  </div>
</EnhancedCodePreview>

## With Helper Text

Radio buttons can include helper text to provide additional information:

<EnhancedCodePreview
  componentName="RadioWithHelperText"
  code={`<Radio 
  value="option1"

  helperText="Delivery in 3-5 business days"
/>`}
>

    value="option1" 
    label="Standard shipping" 
    helperText="Delivery in 3-5 business days"
  />
</EnhancedCodePreview>
