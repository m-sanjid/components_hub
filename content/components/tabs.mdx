# Tabs Component

Tabs components are used to organize content into separate views that can be switched between. This component includes micro-interactions and animations for better user experience.

## Basic Usage

<EnhancedCodePreview
  componentName="Tabs"
  code={`import { Tabs } from '@/data/components/Tabs/Tabs';

const tabs = [
{
value: "tab1",
label: "Tab 1",
content: <p>Content for tab 1</p>,
},
{
value: "tab2",
label: "Tab 2",
content: <p>Content for tab 2</p>,
},
{
value: "tab3",
label: "Tab 3",
content: <p>Content for tab 3</p>,
},
];

<Tabs tabs={tabs} />`}

## With OnChange Handler

Tabs can have an onChange handler to respond to tab changes:

<EnhancedCodePreview
  componentName="TabsWithOnChange"
  code={`import { Tabs } from '@/data/components/Tabs/Tabs';
import { useState } from 'react';

const TabsWithOnChangeExample = () => {
const [activeTab, setActiveTab] = useState("tab1");

const tabs = [
{
value: "tab1",
label: "Tab 1",
content: <p>Content for tab 1</p>,
},
{
value: "tab2",
label: "Tab 2",
content: <p>Content for tab 2</p>,
},
{
value: "tab3",
label: "Tab 3",
content: <p>Content for tab 3</p>,
},
];

return (
<div className="space-y-4">
<Tabs
tabs={tabs}
defaultValue={activeTab}
onValueChange={(value) => {
setActiveTab(value);
console.log("Active tab:", value);
}}
/>
<p className="text-sm text-muted-foreground">
Active tab: {activeTab}
</p>
</div>
);
};`}

>

  <TabsWithOnChangeExample />
</EnhancedCodePreview>
