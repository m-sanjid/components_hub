# Input Component

The `Input` component is a modern, animated, accessible input field that supports:

- Floating labels using Framer Motion
- Left and right icons
- Error and helper text states
- Focus box shadows
- Full width control

## Basic Usage

<EnhancedCodePreview
  componentName="Input"
  code={`import Input from '@/data/components/Input/Input';

// Basic input
<Input />

// With label
<Input label="Username" />

// With helper text
<Input
  label="Password"
  helperText="Must be at least 8 characters"
/>

// With error
<Input 
  label="Email" 
  error="Please enter a valid email"
/>
`}
>
  <div className="flex flex-col gap-4 w-full max-w-md">
    <Input />
    <Input label="Username" />
    <Input 
      label="Password" 
      helperText="Must be at least 8 characters"
    />
    <Input 
      label="Email" 
      error="Please enter a valid email"
    />
  </div>
</EnhancedCodePreview>

## States

Inputs can be in different states:

- Default
- Error
- Disabled
